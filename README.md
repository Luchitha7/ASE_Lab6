# ASE Lab 6 - TDD 

Minimal e-commerce domain built using Test-Driven Development (TDD).
Implements requirements A–F: Product/Catalog, Cart, Inventory reservation, Discount rules, Checkout/payment, and Order persistence.

## Run tests

From this folder:

```bash
python3 -m pytest -q
```

## Project layout

- `src/` contains the implementation
- `src/tests/` contains the test suite (drives the TDD flow)
